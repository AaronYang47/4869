隐私政策

最后更新：2025 年 1 月 1 日

简而言之：当前版本的 4869 不会收集、传输或共享任何个人数据。它没有服务器、账户、遥测、分析或任何网络请求功能。本地保险库数据和偏好设置始终保留在您的 Mac 上。

1. 概述

4869（"本应用"）是一款适用于 macOS 的纯本地文件加密应用。当前版本不包含任何将信息从您的计算机发送到外部服务的代码。

本隐私政策描述了本应用如何处理数据，包括在您的设备上本地处理的数据，以及这些数据如何受到保护。

2. 我们不收集的数据

本应用明确不会收集或处理以下任何信息：

  • 个人信息：绝不会请求或存储任何姓名、电子邮件、电话号码或身份信息。
  • 账户数据：本应用没有账户系统。无需登录、注册或身份验证。
  • 使用分析：不会收集任何遥测、崩溃报告、使用统计或行为数据。
  • 网络数据：本应用不会发起任何网络请求。没有 API 端点、没有服务器、没有云基础设施。
  • 广告数据：本应用不含任何广告，也不参与任何广告网络。
  • 设备信息：不会读取任何硬件标识符、MAC 地址、序列号或系统配置文件。
  • 位置数据：本应用不会访问、请求或存储任何位置信息。

3. 存储在您设备上的数据

以下数据存储在您的 Mac 本地。这些信息绝不会传输到任何地方。

3.1 保险库文件

当您加密文件夹时，本应用会在原始文件夹的位置创建加密保险库数据（一个 .4869 包）。这些数据通过 scrypt 密钥派生和 AES-256-GCM + ChaCha20-Poly1305 加密技术，由您的密码保护。加密过程完成后，本应用不会保留您的明文文件或密码的副本。

3.2 后量子恢复密钥（可选）

如果您启用了后量子恢复功能，系统会生成一份恢复密钥文档，供您保存到自选的位置。本应用不会保留此密钥。如果您同时丢失了密码和恢复密钥，您的数据将永久无法恢复。

3.3 应用偏好设置

本应用使用 macOS 的 UserDefaults 存储少量偏好设置：

  • 临时解锁时长偏好（1、5、15 或 30 分钟）。
  • 当前正在加密的文件夹路径（仅用于崩溃恢复；加密完成后立即清除）。

这些偏好设置不含敏感个人数据，仅存储在您的设备上。

3.4 临时解密缓存

使用临时查看功能时，文件的解密副本会被放置在系统临时目录中，并设置为仅所有者可访问。当查看会话到期、应用正常退出时，或中断后的下次启动时，本应用会清除这些文件。存储硬件和文件系统的行为可能影响物理擦除的彻底性。

4. 您的数据如何受到保护

本应用采用业界领先的加密技术保护您的数据：

  • scrypt 密钥派生（RFC 7914），参数 N=131,072、r=8、p=1，每次密码验证尝试强制占用约 128 MB 内存。
  • 双层认证加密，对每个文件块使用 AES-256-GCM（内层）和 ChaCha20-Poly1305（外层）。
  • HKDF-SHA512，用于域分隔的子密钥派生。
  • 密码学安全的随机数生成，通过 SecRandomCopyBytes 实现。
  • 核心转储保护，通过 setrlimit(RLIMIT_CORE, 0) 防止内存被写入磁盘。
  • 尽力而为的内存处理，使用 memset_s 对可变密钥缓冲区进行清零，并缩短密钥生命周期。由 Swift 或 CryptoKit 内部管理的副本无法保证被覆盖。
  • 安全文件删除，通过随机覆盖、截断和移除来实现。
  • 加密算法自检，在每次应用启动时验证算法完整性。

5. 第三方服务

本应用不使用任何第三方服务。应用运行过程中不涉及任何 SDK、不涉及发起网络调用的库、不涉及分析平台、不涉及广告网络，也不涉及任何云服务提供商。

本应用完全依赖 Apple 内置的框架（CryptoKit、CommonCrypto、Security、Foundation、AppKit），这些框架是 macOS 的组成部分，并遵循 Apple 自身的隐私标准。

6. 儿童隐私

本应用不面向 13 岁以下的儿童。本应用不会故意收集任何人（包括儿童）的个人信息。如果您认为有儿童向本应用提供了个人信息，请放心，本应用没有任何接收、存储或处理此类信息的途径。

7. 您的权利

由于本应用不收集任何数据，以下权利已自然得到满足：

  • 访问权：您的所有数据都在您的设备上。您随时拥有完全的访问权限。
  • 删除权：删除本应用及其保险库文件即可永久移除所有数据。
  • 数据可携带权：您的加密保险库文件是可携带的，可自由移动或复制。
  • 退出权：没有任何需要退出的内容——因为不会收集任何数据。

凭借其零数据收集的设计，本应用符合《通用数据保护条例》（GDPR）、《加州消费者隐私法》（CCPA）以及所有其他主要隐私法规的要求。

8. 数据泄露通知

由于本应用不在任何服务器上存储数据，且不具备网络功能，因此影响本应用的服务器端数据泄露在结构上是不可能的。如果在本应用本身发现漏洞，将在此网站及应用的 GitHub 仓库上予以披露。

9. 本政策的变更

如果本应用的数据处理方式将来发生变化（例如未来版本添加了可选的云同步功能），本隐私政策将相应更新，"最后更新"日期也将反映此变更。您始终可以在此页面上找到本政策的最新版本。
