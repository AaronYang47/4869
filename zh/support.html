⚠️ Important Notice About Password Recovery

If you have forgotten your password and did not set up a post-quantum recovery key, there is nothing we can do to recover your data. This is a fundamental security property of the App — there is no backdoor, no master key, and no server-side override. If you are in this situation, we sympathize, but mathematics cannot be bargained with. Please always set up post-quantum recovery and store your recovery key in a safe, separate location.
